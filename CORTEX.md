# CHH Cortex — Operating contract

## Folder contract

| Path | Purpose |
|------|---------|
| `raw/` | Source material — append-only |
| `wiki/` | Curated knowledge — source-linked |
| `skills/` | Runnable workflows |
| `_system/` | Install, adapters, manifest |
| `TASKS.md` | Shared task list |

## Promotion rules

- Land evidence in `raw/` before wiki promotion
- Every `wiki/` fact needs `Source(s)`
- Curator approves promotions during pilot

## Pilot guardrails

- No PII/PHI in shared vault
- Red-team before paste-ready grant text
- `verify` labels on uncertain claims

See `wiki/partnerships/chh.md` for full pilot dossier.
