# Grant proposal workbench

Each active CHH grant lives under `raw/proposals/<slug>/`.

## Standard files

| File | Role |
|------|------|
| `README.md` | Package index + station |
| `DECISION-CHART.md` | Canonical route, locks, blockers |
| `DONOR-DOSSIER.md` | Must-prove / kill list |
| `REVISION-LOG.md` | Human revision feedback |
| `narrative-skeleton.md` | Section outline |

## Boot command

```
I am working on <slug> — boot the workbench and summarize DECISION-CHART + DONOR-DOSSIER.
```

## Example

`raw/proposals/example-chh-rfp-2026/` — fictional RFP walkthrough (safe to share).
