# jhu-lsri-individual-2026 — Reviewer lens (sanitized)

**Status:** example workbench for CHH Cortex lite  
**Purpose:** Demonstrates DONOR-DOSSIER pattern — not a live submission package

## Must-prove list

1. **Bold, high-risk/high-reward** — not incremental program expansion
2. **AI / data science relevance** — explicit in narrative where applicable
3. **Life sciences scope** — public health / maternal bridge acceptable
4. **Credible follow-on funding** path (NIH, foundations)
5. **JHU faculty team** capability + environment (CHH, CBID)
6. **Budget discipline** — ≤10% external subaward; JHU-hosted work

## Kill list (do not put in application)

1. External NGO as **primary funded implementer** at consortium scale
2. Student listed as **co-PI** (student rules)
3. Conflation with separate large team EoI as same project
4. Claims of **clinical deployment** without IRB frame
5. Unverified conversation corpus statistics
6. PII/PHI in methods description without IRB frame

## Optional-trap list

1. Over-promising **RCT** when pilot/observational fits budget
2. Generic chatbot language without **population-specific** science gap
3. Tooling grant without **health outcome** linkage

## Cortex pattern

Before drafting prose, hold this reviewer lens in `DONOR-DOSSIER.md` inside the grant workbench.
