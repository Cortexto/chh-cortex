# How to use this demo (5–10 minutes)

## Who this is for

Paul, William, Manny, or any CHH staff who wants to see **how files replace chat memory** without touching a live proposal.

## Steps

1. Mount the `chh-cortex` folder in your AI tool.
2. Say: **`I am working on demo-lsri-workbench-2026`**
3. Ask for a **boot summary** — the assistant should cite `DECISION-CHART.md` and `DONOR-DOSSIER.md`, not invent strategy.
4. Open `hypothesis-brief.md` — notice **Fork A vs Fork B** (how Cortex holds forks before locking).
5. Open `REVISION-LOG.md` — see how human feedback becomes durable memory.
6. Ask the assistant to **decode one portal-style question** using the workflows guide (field job / reviewer check / trap).

## What is fake on purpose

- Deadlines and names marked `[example]`
- No live capture-log paths or internal agent ledgers
- No external partner budget politics — only academic reviewer shape

## After the demo

Copy `raw/proposals/_template-chh-rfp/` to `raw/proposals/your-grant-slug/` and fill for a real RFP.

Read **`CHH-ROADMAP.md`** at repo root for email population and phased rollout.
